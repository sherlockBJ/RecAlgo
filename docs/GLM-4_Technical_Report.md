## 2. 预训练

### 数据规模
- **总量**: 10万亿+ tokens (GLM-4), 15万亿 tokens (GLM-4-9B-0414)
- **语言分布**: 主要中英文 + 24种其他语言
- **数据类型**: 包含大量推理类合成数据

### 架构特点
- **精度**: BF16 (bfloat16)
- **长文本**: 支持 YaRN 位置编码外推
- **原生上下文**: 32K，扩展至 128K/1M

---

## 3. 对齐训练 (Post-Training)

采用**多阶段后训练**流程：

1. **监督微调 (SFT)**: 高质量指令数据
2. **人类偏好对齐**: 基于人类反馈学习
3. **拒绝采样 (Rejection Sampling)**: 提升输出质量
4. **强化学习**: 进一步优化对话能力

| 基准 | GLM-4V-9B | GPT-4V | Claude 3 Opus |
|------|-----------|--------|---------------|

- MMLU、GSM8K、MATH、BBH、GPQA、HumanEval: **接近或超越 GPT-4**

- 中文 AlignBench: **超越 GPT-4**
- 指令跟随 IFEval: **接近 GPT-4-Turbo**

## 5. GLM-4 All Tools 工具能力

GLM-4 All Tools 版本具备**自主工具调用**能力：

| 工具类型 | 功能 |
|----------|------|
| **网页浏览** | 实时搜索与信息获取 |
| **Python 解释器** | 代码执行与计算 |
| **文生图模型** | 图像生成 |
| **自定义函数** | 用户定义工具调用 |

**特点**: 模型能够理解用户意图，自主决定何时使用哪些工具。

## 8. 技术要点总结

### 8.1 核心创新
1. **超长上下文**: 原生支持128K，扩展至1M tokens
2. **多语言能力**: 26种语言支持
3. **工具集成**: All Tools 版本实现自主工具调用
4. **多模态扩展**: GLM-4V 支持高分辨率视觉理解

### 8.2 训练优化
1. 大规模高质量预训练数据（10T+ tokens）
2. 多阶段对齐训练流程
3. 推理类合成数据增强
4. YaRN 位置编码支持长文本外推

### 8.3 部署支持
- Transformers >= 4.46.0
- vLLM 高效批量推理
- 多平台模型分发（HuggingFace、ModelScope）
