# 二、QwQ (Qwen with Questions)

## 模型信息
- **版本**: QwQ-32B-Preview
- **参数量**: 32B
- **定位**: 深度推理模型
- **官方博客**: [qwenlm.github.io/blog/qwq-32b-preview](https://qwenlm.github.io/blog/qwq-32b-preview/)

---

## 1. 核心特点

### 推理方式
- 强调**反思性问题解决**
- 对问题保持"真正的好奇和怀疑"
- 是思考型推理者，而非快速响应系统

### 设计理念
- 名称发音类似 "quill"（羽毛笔）
- 体现"先质疑假设，再得出结论"的哲学

---

## 2. 评测结果

| 基准 | QwQ-32B-Preview | 说明 |
|------|-----------------|------|
| **GPQA** | 65.2% | 研究生级科学推理 |
| **AIME** | 50.0% | 中学数学竞赛 |
| **MATH-500** | 90.6% | 多样化数学题 |
| **LiveCodeBench** | 50.0% | 真实编程场景 |

---

## 3. 已知限制

| 限制类型 | 描述 |
|----------|------|
| **语言混合** | 可能意外切换语言 |
| **循环推理** | 可能产生冗长无结论的回答 |
| **安全措施** | 需要增强的安全机制 |
| **能力不均** | 数学/编程强，常识推理弱 |

---

# 三、技术对比总结

## Qwen2.5 vs QwQ

| 特性 | Qwen2.5 | QwQ |
|------|---------|-----|
| **定位** | 通用大模型 | 推理专用 |
| **最大规格** | 72B | 32B |
| **核心优势** | 多语言、长上下文 | 深度推理、反思 |
| **适用场景** | 通用任务 | 数学、编程 |

## 与其他推理模型对比

| 模型 | MATH-500 | AIME | GPQA |
|------|----------|------|------|
| **QwQ-32B** | 90.6% | 50.0% | 65.2% |
| **DeepSeek-R1** | 97.3% | 79.8% | - |
| **OpenAI-o1** | ~96% | ~80% | - |
